Esperando para cambiar...

En esta actividad, aprenderás a crear un temporizador que cambiará el contenido de un elemento después de un tiempo determinado.

Crea un archivo HTML con un párrafo que diga "Esperando para cambiar..."

Utilice JavaScript para cambiar el texto del párrafo a "Texto cambiado después de 3 segundos" después de 3 segundos de haber cargado la página

Guarda los archivos como actividad14.html y actividad14.js.
